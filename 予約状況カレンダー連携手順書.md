# 予約状況カレンダー ＆ お問い合わせフォームLINE通知 連携手順書 (LINE Messaging API対応版)

この手順書は、ホームページ上の「予約空き状況カレンダー」の同期に加え、お客様が「お問い合わせフォーム」から見積もり依頼を送信した際に、**店主様のLINEへリアルタイム通知** が届くようにするための設定マニュアルです。

※LINE Notifyがサービス終了したため、現在サポートされている **LINE Messaging API (LINE Bot)** を使って店主様個人のLINEへ通知する仕組みを導入します。

---

## 設定手順（4ステップ）

### ステップ 1. LINE Developers から「トークン」と「ユーザーID」をコピーする

店主様が現在開いていただいている **LINE Developers（または LINE Official Account Manager）** の画面から、連携に必要な2つの接続キーを取得します。

#### ①「チャネルアクセストークン（長期）」の取得
1. 管理画面のメニュー（または上部タブ）から、 **[Messaging API設定]** を開きます。
2. 画面の一番下までスクロールし、 **「チャネルアクセストークン（長期）」** グループを見つけます。
3. **[発行]** ボタンをクリックします。
4. 発行された**非常に長い英数字の文字列（アクセストークン）をコピー**して、メモ帳等に一時保存します。

#### ②「あなたのユーザーID」の取得
1. 管理画面のメニューから、 **[チャネル基本設定]** タブを開きます。
2. 画面を一番下までスクロールし、 **「あなたのユーザーID (Your user ID)」** グループを見つけます。
3. **`U` から始まる約33文字の文字列（ユーザーID）をコピー**して、メモ帳等に一時保存します。
   *   *(※これはLINEにログインする際のメールアドレスやIDとは異なり、開発者用の識別コードです)*

#### ③ 公式アカウントを友だち追加する
1. **[Messaging API設定]** タブに戻り、画面中央にある **QRコード** をご自身のスマートフォンのカメラで読み取ります。
2. 表示された公式アカウントを**友だち追加**します。
   *   *※友だち追加をしていないと、公式アカウントから店主様のLINEへ通知を送ることができません。*

---

### ステップ 2. 中継する「GASプログラム」を作成し、キーを設定する

1. [Googleドライブ](https://drive.google.com) にログインします。
2. 画面左上の **[新規] ＞ [その他] ＞ [Google Apps Script]** をクリックします。
3. 開いたプログラム編集画面のコードを全て消去し、代わりに**以下のコードを丸ごとコピーして貼り付けます。**

```javascript
// ==========================================================================
// 【設定項目 1】ステップ 1 で取得した LINE チャネルアクセストークン（長期）
// ==========================================================================
const LINE_CHANNEL_ACCESS_TOKEN = 'ここにコピーしたチャネルアクセストークンを貼り付け';

// ==========================================================================
// 【設定項目 2】ステップ 1 で取得した あなたの LINE ユーザーID
// ==========================================================================
const LINE_USER_ID = 'ここにコピーしたあなたのユーザーIDを貼り付け';

// ==========================================================================
// 【設定項目 3】連携する Google カレンダー ID
// ==========================================================================
const CALENDAR_ID = 'bd79e71053336caac06328d0534352d20c4fbe3e282d7b2d82ecef8f69969750@group.calendar.google.com';


// --------------------------------------------------------------------------
// 1. カレンダー空き状況（時間数）を取得する処理 (GET)
// --------------------------------------------------------------------------
function doGet() {
  try {
    const calendar = CalendarApp.getCalendarById(CALENDAR_ID);
    if (!calendar) {
      return createJsonResponse({ error: "カレンダーが見つかりません。" });
    }
    
    const today = new Date();
    // 今日から翌々月末までの予定を取得（LPに表示する範囲：3ヶ月分）
    const endRange = new Date(today.getFullYear(), today.getMonth() + 3, 1);
    const events = calendar.getEvents(today, endRange);
    
    const blockedDates = {};
    events.forEach(event => {
      let durationHours = 0;
      
      // 終日予定（定休日や臨時休業など）は無条件で8時間（満員）としてカウント
      if (event.isAllDayEvent()) {
        durationHours = 8.0; 
      } else {
        const diffMs = event.getEndTime() - event.getStartTime();
        durationHours = diffMs / (1000 * 60 * 60); // ミリ秒から時間数（Hours）に変換
      }
      
      const start = event.getStartTime();
      const y = start.getFullYear();
      const m = String(start.getMonth() + 1).padStart(2, '0');
      const d = String(start.getDate()).padStart(2, '0');
      const dateKey = `${y}-${m}-${d}`;
      
      if (!blockedDates[dateKey]) {
        blockedDates[dateKey] = 0.0;
      }
      blockedDates[dateKey] += durationHours;
    });
    
    return createJsonResponse(blockedDates);
      
  } catch (e) {
    return createJsonResponse({ error: e.toString() });
  }
}

// --------------------------------------------------------------------------
// 2. ホームページからの申し込みフォームデータを受け取り、LINE通知する処理 (POST)
// --------------------------------------------------------------------------
function doPost(e) {
  try {
    const params = e.parameter;
    
    // フォーム送信のリクエストであるか確認
    if (params.type === 'submit_form') {
      const name = params.name || '';
      const tel = params.tel || '';
      const email = params.email || '';
      const service = params.service || '';
      const workArea = params.workArea || '';
      const visitDate = params.visitDate || '';
      const message = params.message || '';
      
      // LINEへ送信するメッセージ本文を組み立て
      const lineMessage = 
        "【マスト便利屋】お見積もり依頼\n" +
        "------------------\n" +
        "■お名前: " + name + " 様\n" +
        "■お電話: " + tel + "\n" +
        "■メール: " + email + "\n" +
        "■サービス: " + service + "\n" +
        "■作業場所: " + workArea + "\n" +
        "■希望作業日: " + visitDate + "\n" +
        "------------------\n" +
        "■お問い合わせ内容:\n" + message;
        
      // LINEへ通知
      sendToLineBot(lineMessage);
      
      return createJsonResponse({ status: 'success', message: '送信に成功しました。' });
    }
    
    return createJsonResponse({ status: 'error', message: '無効なリクエストタイプです。' });
    
  } catch (err) {
    return createJsonResponse({ status: 'error', message: err.toString() });
  }
}

// LINE Messaging API を使ってメッセージをプッシュ送信する処理
function sendToLineBot(message) {
  if (!LINE_CHANNEL_ACCESS_TOKEN || !LINE_USER_ID || LINE_CHANNEL_ACCESS_TOKEN.indexOf('ここにコピーした') > -1) {
    Logger.log("LINE トークンまたはユーザーIDが未設定です。");
    return;
  }
  
  const url = "https://api.line.me/v2/bot/message/push";
  const postData = {
    to: LINE_USER_ID,
    messages: [
      {
        type: "text",
        text: message
      }
    ]
  };
  
  const options = {
    method: "post",
    contentType: "application/json",
    headers: {
      Authorization: "Bearer " + LINE_CHANNEL_ACCESS_TOKEN
    },
    payload: JSON.stringify(postData),
    muteHttpExceptions: true
  };
  
  const response = UrlFetchApp.fetch(url, options);
  Logger.log(response.getContentText());
}

// CORS制限を回避するJSONレスポンスを生成する共通関数
function createJsonResponse(data) {
  return ContentService.createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}
```

4. 4行目と 9行目のプレースホルダー部分をご自身の **「チャネルアクセストークン」** と **「ユーザーID」** に書き換えます。
5. ツールバーの **「保存」アイコン（フロッピーディスクのマーク）** をクリックして保存します。

### ステップ 3. プログラムを「ウェブアプリ」として公開する

1. 画面右上の青い **[デプロイ] ＞ [新しいデプロイ]** をクリックします。
2. 種類の選択（歯車のマーク）から **[ウェブアプリ]** を選択します。
3. 各設定を以下のように指定します：
   *   **ウェブアプリを実行**: **「自分（店主様のアカウント）」**
   *   **アクセスできるユーザー**: **「全員（匿名ユーザーを含む）」** （※必ず全員にしてください）
4. **[デプロイ]** ボタンをクリックし、アクセス承認が表示された場合はアカウントへの権限を許可（詳細 ＞ 安全ではないページに移動 ＞ 許可）します。
5. デプロイ完了画面に表示される **「ウェブアプリのURL」** をコピーします。

---

### ステップ 4. コピーしたURLをホームページの `main.js` に貼り付ける

1. ホームページのフォルダ内にある `main.js` ファイルを開きます。
2. 233行目付近にある以下の記述を探します。
   ```javascript
   const GOOGLE_CALENDAR_GAS_URL = 'https://script.google.com/macros/s/XXXXXX/exec';
   ```
3. すでに設定されているURLを、今回コピーした **新しいウェブアプリのURL** に差し替えて保存します。これで全ての設定が完了です！

---

## 運用時のルール
*   **「○・△・×」の自動切り替え基準**:
    *   **○（空きあり）**: Googleカレンダーの予定の合計が **3時間未満** の日。
    *   **△（残りわずか）**: 予定の合計が **3時間以上〜8時間未満** の日。
    *   **×（空きなし）**: 予定の合計が **8時間以上**、または時間設定なしの「終日予定」がある日。
*   **お問い合わせ送信時の動作**:
    ホームページのフォームでお客様が入力して「見積もりを申し込む」ボタンを押すと、即座に公式アカウントから店主様個人のLINEへメッセージとして送信データ一式が通知されます。
